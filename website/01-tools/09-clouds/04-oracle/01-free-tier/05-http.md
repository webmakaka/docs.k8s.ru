# Доступ к облаку Oracle по http

```
$ python3 -m http.server 8080
```

<br/>

Networking > Virtual Cloud Networks > VCN_LINUX_DISPLAY_NAME > Security List Details

<br/>

```
Source Port Range -> ALL
Destination Port Range -> 8080
```
